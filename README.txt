MENU MENACE - folder version

Open index.html in Safari, Chrome, Edge, or another normal browser.

Files in this folder:
- index.html = the page structure
- style.css = the look/colors/layout
- script.js = the actual game buttons and logic

Important iPad/iPhone note:
If you tap the file and it opens in a Files preview, JavaScript may not run. If the buttons do nothing, share/open index.html in Safari or Chrome from the unzipped folder.

Save export/import:
Use Settings > Export save. The downloaded file name uses your restaurant name and chef name, like:
Little Pizzeria - Maria.menu-menace-save.json

Use Settings > Import save to load that file again later.
