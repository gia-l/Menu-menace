MENU MENACE - BUILD-YOUR-OWN TINY RESTAURANT

How to open
1. Unzip the folder.
2. Open index.html in Safari, Chrome, Edge, or another real browser.
3. Avoid opening it only in a Files/Drive preview, because previews can show the page but block the JavaScript buttons.

What this version starts with
This build uses a fresh save key, so it starts clean instead of pulling in old pizza/grilled-cheese test data from earlier builds.

The only starter example is spaghetti:
- flavors: savory / umami, sweet / sweetness, fresh / freshness
- ingredients: pasta, tomato sauce, herbs
- dish: spaghetti

You can remove or edit all of those. They are just examples.

Core loop
1. Build flavor profiles. A flavor has an adjective and a noun form, like savory + umami, sweet + sweetness, magical + magic, spicy + heat.
2. Build ingredients. Give each ingredient flavor amounts.
3. Build dishes. Pick base ingredients and optional ingredients.
4. Invite a customer at the Counter.
5. Go to the Kitchen, tap ingredients into the pot, then give the dish to the customer.
6. Customers comment on the food. Coins and XP appear under the dialogue instead of inside what they say.

Trends
Today's trend only uses flavors that exist in your restaurant. No spicy trend unless you create spicy.
A dish gets a trend bonus if its final flavor profile is at least 50% of the trending flavor.
Manual trend rerolls cost gems or a lot of coins. The cost goes up each time until the next timed trend refresh.

Coins, XP, gems, and slots
Customers pay coins. More ingredients usually means more possible pay, but mistakes reduce the final payment.
Serving customers gives XP. Leveling up gives gems equal to your new level.
Menu slots limit how many dishes can be active. Buy more slots with coins.

Cosmetics
Cosmetics now have their own tab. Spend gems to unlock restaurant themes. Cosmetics only change the look, not the rules.

Editing
In the Build tab, use the Edit buttons beside flavors, ingredients, and dishes to revise things you already made.

Saving
The game autosaves in your browser.
Use Export save to download a backup. The filename uses your restaurant name and chef name, for example:
Little Pizzeria - Maria.menu-menace-save.json

Use Import save to restore that backup later or move it to another device.
