MENU MENACE - scratch build version

Open index.html in Safari, Chrome, Edge, or Firefox after unzipping this folder.
Do not play it inside the iPad/iPhone Files preview if buttons act weird.

What changed in this version:
- The game starts with only one plain starter example: spaghetti.
- Starter flavors are only savory, sweet, and fresh because the spaghetti example uses them.
- Trends are chosen only from the flavor profiles that exist in your restaurant.
- If you delete spicy or never create spicy, spicy cannot become today's trend.
- Dishes, dish categories, ingredients, and ingredient categories are collapsible.
- You can delete starter dishes, ingredients, and flavors and build from scratch.
- Settings includes a completely blank start option.
- Exported saves use your restaurant name and chef name in the file name.

Suggested first custom build:
1. Add a flavor profile, like spicy / heat.
2. Add ingredients and assign flavor amounts.
3. Add a dish and choose its base and optional ingredients.
4. Invite a customer and cook.
